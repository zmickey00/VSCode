This repository contains Zhiyu Zhang's code for two courses:

  1. Stochastic Process (MATH541)
  2. Microeconometrics (ECON690)

Both courses require coding simulations and data analysis. 

  Most coding for MATH541 is about simulating stochastic process,
  Most coding for ECON690 is about understanding statistical methods including MLE, Bootstrap, OLS, GMM, etc. 

Each individual project (most them are homework questions) are contained in folders under MATH 541, ECON 690. 
Each homework folder contains the original code, a PDF report summarizing the code and answering the homework quesitons, and the latex file that produces the PDF.
